## 🤖 Currently Building

### 🧠 DocuMind RAG — Agentic AI Document Q&A System
A production-grade **Agentic Retrieval-Augmented Generation (RAG)** system with autonomous decision-making capabilities.

**🔗 Live Project:** [github.com/2salman-19/documind-rag](https://github.com/2salman-19/documind-rag)

**Key Features:**
- 🤖 **AI Agents** with LangChain & LangGraph (ReAct pattern)
- 🔍 **Hybrid Search** (BM25 + Vector) with Cross-Encoder Re-ranking
- ⚡ **Real-Time Streaming** via Server-Sent Events (SSE)
- 🧮 **Dynamic Tool Routing** (RAG Search, Calculator, Web Search)
- 💭 **Thought Visualization** — see the agent's reasoning live
- 📦 **Supabase pgvector** for scalable vector storage
- 🚀 **Groq (Llama-3.3-70B)** for ultra-fast inference

**Tech Stack:** Python · FastAPI · LangChain · LangGraph · Supabase · Groq · SSE · React-style Vanilla JS
